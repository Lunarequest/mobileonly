My take on mobile first programing

When we hear mobile first programing we often think of apks and java/kothlin. All that was true for me until I saw a youtube video where joma tech made a website only using his iphone.There were some places I felt he lost the spirit of the challenge like using a e2c instance. So I took it upon myself to do this while avoiding breaking away from the spirit

This website your reading was made using only my phone. No e2c instance just termux and a archlinux evniorment provided by anlinux.my main aim when I did this was to show that programing isn't inaccessable. All you need is a smartphone,perserverance and free time
